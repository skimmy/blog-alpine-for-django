Event Listener

Code

<div x-data="" id="content">
    <!-- @click is shortcut for x-on:click -->
    <button @click="console.log('Clicked!')">Click to Log!</button>
</div>

Reactive variables

Clicks:

Code

<div x-data="{count: 0}">
    <button @click="count++">Click: </button>
    Clicks: <strong x-text="count"></strong>
</div>

Model binding


The input text contains:

Code

<div x-data="{text: ''}">
    <input type="text" x-model="text"><br>
    The input text contains: <span style="color: blue" x-text="text"></span>
</div>

Attribute binding

Code

<div x-data="{ids: [2, 5, 3]}">
    <template x-for="id in ids">
        <p>
            <span x-text="`${id}: `"></span><input type="text" x-bind:name="`input_{$i}`" placeholder="">
        </p>
    </template>
</div>
